## Overview

Upload an extension as a ZIP file. The ZIP must contain an `extension.json` manifest at the root level or inside a single wrapper directory. The extension is validated and installed to `~/.pocketpaw/extensions/<id>/`.

Built-in extensions cannot be overwritten — only external extensions can be uploaded or replaced.

## Request Body

Send as `multipart/form-data`:

<ParamTable type="body">
  <Param name="file" type="file" required>
    ZIP file containing the extension. Must have a `.zip` extension. Maximum
    size: 50 MB.
  </Param>
</ParamTable>

## Response

Returns the updated extensions list (same format as `GET /extensions`).

<ResponseField name="extensions" type="array">
  Array of extension summaries with the newly installed extension included.
</ResponseField>
<ResponseField name="errors" type="array">
  Any loading errors from the extension registry.
</ResponseField>

## Error Responses

| Status | Reason                                                                         |
| ------ | ------------------------------------------------------------------------------ |
| 400    | Not a `.zip` file, invalid ZIP, no `extension.json` found, or invalid manifest |
| 403    | Extension tokens cannot access management APIs                                 |
| 409    | Cannot overwrite a built-in extension                                          |
| 413    | File exceeds 50 MB size limit                                                  |

<RequestExample>
<Tabs items={["cURL", "JavaScript", "Python"]}>
  <Tab title="cURL">
    ```bash
    curl -X POST "http://localhost:8888/api/v1/extensions/upload" \
      -H "Authorization: Bearer <token>" \
      -F "file=@my-extension.zip"
    ```
  </Tab>
  <Tab title="JavaScript">
    ```javascript
    const formData = new FormData();
    formData.append("file", fileInput.files[0]);

    const response = await fetch("/api/v1/extensions/upload", {
      method: "POST",
      headers: { "Authorization": "Bearer <token>" },
      body: formData,
    });
    const data = await response.json();
    console.log(data.extensions);
    ```

  </Tab>
  <Tab title="Python">
    ```python
    import requests

    with open("my-extension.zip", "rb") as f:
        response = requests.post(
            "http://localhost:8888/api/v1/extensions/upload",
            headers={"Authorization": "Bearer <token>"},
            files={"file": ("my-extension.zip", f, "application/zip")},
        )
    print(response.json())
    ```

  </Tab>
</Tabs>
</RequestExample>
