## Overview

Delete an externally-installed extension by removing its directory from `~/.pocketpaw/extensions/`. Built-in extensions cannot be deleted — disable them instead via `POST /extensions/{id}/enabled`.

## Path Parameters

<ParamTable type="path">
  <Param name="extension_id" type="string" required>
    The unique ID of the extension to delete.
  </Param>
</ParamTable>

## Response

<ResponseField name="status" type="string">
  `"ok"` on success.
</ResponseField>
<ResponseField name="deleted" type="string">
  The ID of the deleted extension.
</ResponseField>

## Error Responses

| Status | Reason                            |
| ------ | --------------------------------- |
| 403    | Cannot delete built-in extensions |
| 404    | Extension not found               |
